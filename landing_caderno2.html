O que você vai aprender a ler

Código que escolhe

Você lê o if no papel, rastreia a condição e descobre qual bloco executa — antes de abrir qualquer compilador.

StatusLeon.cs — Lição 4, exercício rastreado no papel
int vida = 45;
 
if (vida > 80)
{
    Console.WriteLine("Status: ÓTIMO");
}
else if (vida > 50)
{
    Console.WriteLine("Status: BOM");
}
else if (vida > 20)
{
    Console.WriteLine("Status: PERIGO");
}
else
{
    Console.WriteLine("Status: CRÍTICO");
}
Saída — você aprende a prever isso no papel
Status: PERIGO
← vida=45: 45>80 false → 45>50 false → 45>20 true → terceiro bloco executa

Senão — else
O caminho alternativo. Tabela de rastreamento com dois cenários. Um dos dois blocos sempre executa — nunca os dois ao mesmo tempo.

Vários caminhos — else if
Encadeamento de condições. Regra de ouro: só o primeiro true executa. Exercício: sistema de classificação de munição com 4 níveis.
