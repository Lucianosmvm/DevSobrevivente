Antes de digitar, você rastreia no papel. O Caderno 3 te mostra o while e o for impresso, linha por linha, e te ensina a simular a execução numa tabela.

Lição 3 — Loop for no papel
// Onda de inimigos — Lição 5
for (int onda = 1; onda <= 3; onda++)
{
  for (int i = 1; i <= 4; i++)
  {
    Console.WriteLine("Onda " + onda + " — Inimigo " + i);
  }
}
